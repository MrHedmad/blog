My tools #

Some of my favourite day-to-day tools include:

  • Arch Linux

    Just to boast that I use Arch. In actuality, it is just Manjaro which needs not be configured for a day before starting to use it. It also ships with KDE plasma, which is my favourite desktop. Arch is just leagues better than other flavours of Linux thanks to pacman and the official and unofficial repositories. Being a rolling release is also a must for me. I think my old Ubuntu systems broke at every release update.

  • NeoVim

    As my source code editor, I choose NeoVim with VsCode as the backup alternative. The ability to use vim shortcuts (even though I only know of a handful of them) is priceless to me. As a plus, I also get to use my editor without leaving the terminal.

  • RStudio

    To edit R source there is little better than RStudio. RStudio server can also be installed on a remote machine to let you work remotely from the desktop. It is not the best IDE, but the best when it comes to R.

  • The Corne keyboard

    After I bought this keyboard I cannot go back to anything else. It is expensive to purchase, looks amazing, can be configured to your liking and grabs everyone’s attention when they walk in the lab. The keys are Cherry Whites with blank white keycaps (to force me to touch type). It is programmed with via, so it sends the canonical en-US keycodes but with the Hands Down Reference layout.

    I like to think of myself as a keyboard goblin.
